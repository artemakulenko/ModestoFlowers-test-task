## Локальный запуск:

- Клонировать репозиторий
  ```bash
  $ git@github.com:artemakulenko/Test.git
  ```
- Перейти в директорию проекта и установить зависимости

  ```bash
  $ cd Test && npm install
  ```

- Запустить приложение в режиме разработки

  ```bash
  npm run dev
  ```

- Собрать и запускать
  ```bash
  npm run build && npm run preview
  ```
